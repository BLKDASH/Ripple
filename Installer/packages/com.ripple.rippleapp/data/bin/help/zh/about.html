凌波

跨平台串口调试工具，基于 Qt 6 / QML / C++ 构建。轻量、高效、开箱即用。

核心功能

功能说明
高速通信波特率支持 1200 ~ 921600（可自定义），独立线程 I/O 确保 UI 不卡顿
HEX 收发文本 / HEX 双模式发送与接收，实时切换
自动日志按日自动记录接收数据，支持文本与 HEX 格式
快捷发送6 个可配置按钮，JSON 导入导出，一键发送常用指令
主题切换明暗双主题，手动切换并自动记忆
多语言支持 English / 简体中文，运行时切换

技术栈

  • Qt 6 — 跨平台 UI 框架
  • QML — 声明式 UI，Fluid 动画
  • C++ 17 — 高性能后端逻辑
  • QSerialPort — 串口通信
