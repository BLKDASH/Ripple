Quick Send Panel

The quick-send panel provides 15 configurable slots that let you fire frequently-used commands with a single click.

JSON Format

Each slot is a JSON object with three fields:

FieldTypeDescription
namestringOptional label displayed next to the input field
commandstringThe payload to send. In HEX mode, use space-separated hex bytes, e.g. "01 02 FF"
hexbooltrue to send as hex bytes, false to send as raw text

Template

[
  { "name": "", "command": "AT",       "hex": false },
  { "name": "", "command": "AT+GMR",   "hex": false },
  { "name": "", "command": "01 02 FF", "hex": true  },
  { "name": "", "command": "",         "hex": false },
  { "name": "", "command": "",         "hex": false },
  { "name": "", "command": "",         "hex": false },

Toolbar

IconAction
📂Load configuration from a .json file
