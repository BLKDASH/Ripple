Quick Send Grid

The quick-send panel provides 6 configurable buttons that let you fire frequently-used commands with a single click.

JSON Format

Each slot is a JSON object with two fields:

  • command (string) — the payload to send. In HEX mode this should be space-separated hex bytes, e.g. "01 02 FF".
  • hex (bool) — true to send the command as hex bytes, false to send it as raw text.

Template (6 slots)

[
  { "command": "AT",      "hex": false },
  { "command": "AT+GMR",  "hex": false },
  { "command": "",        "hex": false },
  { "command": "",        "hex": false },
  { "command": "",        "hex": false },
  { "command": "",        "hex": false }
]

Use Load / Save to import or export the grid as a .json file.
