Quick Send Grid

The quick-send panel provides 6 configurable buttons that let you fire frequently-used commands with a single click.

JSON Format

Each slot is a JSON object with two fields:

FieldTypeDescription
commandstringThe payload to send. In HEX mode, use space-separated hex bytes, e.g. "01 02 FF"
hexbooltrue to send as hex bytes, false to send as raw text

Template

[
  { "command": "AT",      "hex": false },
  { "command": "AT+GMR",  "hex": false },
  { "command": "01 02 FF", "hex": true  },
  { "command": "",        "hex": false },
  { "command": "",        "hex": false },
  { "command": "",        "hex": false }
]

Import / Export

Use the Load / Save buttons to export your configuration as a .json file — handy for migrating between machines or backing up your presets.
