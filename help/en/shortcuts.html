File Loading

Use the menu Send Area → Load to Send to load a file into the send pane. The mode is selected automatically based on file type:

File TypeLoading Mode
Files ending with .binLoaded as HEX automatically
Text filesLoaded as text
💡 In HEX mode, file contents are automatically converted to space-separated hex byte sequences.
