Auto Log

Auto-log records all incoming serial data to daily log files so you never lose received data.

How it works

  1. Go to Settings and enable "Auto log".
  2. Choose a folder — NOT a file. All logs will be stored there.
  3. When the serial port opens, a new log file is created (or opened) with the name yyyy-MM-dd.log (e.g. 2026-06-19.log).
  4. All incoming data is appended to today's file.
  5. When the port closes, the log file is closed.

Notes

  • Text data is logged as UTF-8. Binary data (more than 5% non-UTF-8 bytes) is logged as a hex preview.
  • If a session spans midnight, data continues writing to the previous day's file. Re-open the port to start a new file.
  • A dashed ring appears on the status-bar indicator when auto-log is active.
