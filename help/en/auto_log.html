Auto Log

Auto-log records all incoming serial data to daily log files in real time, so you never lose received data.

Getting Started

  1. Open CWY → Settings and enable "Auto log".
  2. Click "Browse" to choose a folder (not a file) — all logs will be stored there.
  3. Open the serial port. Logging starts automatically.

File Naming

Log files are named by date: yyyy-MM-dd.log, e.g. 2026-06-19.log. Data from the same day is appended to the same file.

Log Format

Data TypeLogged As
Text dataUTF-8 with [HH:mm:ss.zzz] timestamps
Binary dataHEX preview (auto-detected when >5% non-UTF-8 bytes)
⚠️ If a session spans midnight, data continues writing to the previous day's file. Re-open the port to start a new file.

When auto-log is active, a ring appears on the status bar indicator 📋.
