Ripple

A cross-platform serial debugging tool built with Qt 6 / QML / C++.

Features

FeatureDescription
High-speed I/OBaud rates from 1200 to 921600 (custom supported), dedicated I/O thread keeps UI smooth
HEX supportText / HEX dual-mode send & receive, switch in real time
Auto-logDaily log files with text & HEX formats
Quick send6 configurable buttons with JSON import/export for one-click commands
ThemesDark & light themes, manual toggle with persistence
Multi-languageEnglish & Chinese, switch at runtime

Tech Stack

  • Qt 6 — cross-platform UI framework
  • QML — declarative UI with fluid animations
  • C++ 17 — high-performance backend logic
  • QSerialPort — serial communication
