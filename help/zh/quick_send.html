快捷发送面板

快捷发送面板提供 6 个可配置按钮，一键发送常用指令。

JSON 格式

每个槽位是一个包含两个字段的 JSON 对象：

  • command（字符串）— 要发送的内容。HEX 模式下为空格分隔的十六进制字节，如 "01 02 FF"。
  • hex（布尔值）— true 表示以 HEX 字节发送，false 表示以原始文本发送。

模板（6 个槽位）

[
  { "command": "AT",      "hex": false },
  { "command": "AT+GMR",  "hex": false },
  { "command": "",        "hex": false },
  { "command": "",        "hex": false },
  { "command": "",        "hex": false },
  { "command": "",        "hex": false }
]

使用 Load / Save 按钮导入或导出 .json 配置文件。
