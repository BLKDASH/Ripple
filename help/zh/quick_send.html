快捷发送面板

快捷发送面板提供 6 个可配置按钮，一键发送常用指令，无需重复输入。

JSON 格式

每个槽位是一个 JSON 对象，包含两个字段：

字段类型说明
commandstring要发送的内容。HEX 模式下为空格分隔的十六进制字节，如 "01 02 FF"
hexbooltrue 以 HEX 字节发送，false 以原始文本发送

模板

[
  { "command": "AT",      "hex": false },
  { "command": "AT+GMR",  "hex": false },
  { "command": "01 02 FF", "hex": true  },
  { "command": "",        "hex": false },
  { "command": "",        "hex": false },
  { "command": "",        "hex": false }
]

导入导出

使用面板上的 Load / Save 按钮，可将配置导出为 .json 文件，方便在不同设备间迁移或备份。

💡 建议为常用的 AT 指令、调试命令预设好快捷按钮，可大幅提升调试效率。
